NOT A FORUM Whatever you ask there must have a specific answer; hence you cannot ask questions that merely seeking an opinion (“Why do people hate PHP?”) or is subjective in nature (“Am I a good programmer”). It’s also not a place to debate, or to push an agenda. This is why troubleshooting questions (or “why doesn’t my code work>”) questions are usually not suitable. Code don’t work for many reasons, and sometimes it’s not the code itself that is problematic but perhaps an environment variable or a configuration setting somewhere. Stack Overflow is not suitable for this kind of back and forth.
